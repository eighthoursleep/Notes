## 安装Node.js环境

请[下载](https://nodejs.org)LTS版本。

如何确认node环境已经安装？执行命令

```
npm -h
```

如果能看到npm的命令说明、版本号，以及安装路径等信息，说明已经安装过了。



## 安装TypeScript环境

下载安装命令：

```
npm install -g typescript
```

Typescript编译器版本查看命令：

```
tsc -v
```

如果版本可以成功查看，说明安装成功。



## 下载安装LayaAirIDE

请移步[Laya官网](https://ldc2.layabox.com/)查找

打开LayaAirIDE后新建项目都是傻瓜操作，无需讲解。
